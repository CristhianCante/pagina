Introducción a PyQGIS

Automatizando tareas y creando herramientas personalizadas en QGIS con Python.

1. ¿Qué es PyQGIS?

PyQGIS es la interfaz de programación de aplicaciones (API) de Python para QGIS. En esencia, es la puerta de entrada para controlar y extender las funcionalidades de QGIS mediante código. Permite automatizar tareas repetitivas, crear herramientas personalizadas y desarrollar soluciones GIS innovadoras que van más allá de las funciones predefinidas en la interfaz gráfica.

1.1. Preparando el Entorno: La Consola de Python

QGIS integra una "Consola de Python" que funciona como un laboratorio interactivo donde se pueden probar comandos de PyQGIS de manera inmediata. Para acceder a ella, ve al menú Complementos > Consola de Python.

2. Accediendo a Capas y Proyectos

Para que PyQGIS pueda realizar operaciones, primero necesita acceder a los datos cargados en QGIS. El punto de partida es el objeto iface, que actúa como un "control remoto" para la interfaz de QGIS. Para interactuar con las capas del proyecto actual, usamos QgsProject.instance().

2.1. Carga de Capas Vectoriales

Podemos cargar capas vectoriales (como Shapefiles, GeoJSON, etc.) de forma programática. El método más común es crear un objeto QgsVectorLayer y luego añadirlo al proyecto.

Ejemplo: Cargar una capa de barrios de la localidad de Usaquén.

from qgis.core import QgsVectorLayer, QgsProject

# ¡Importante! Reemplaza esto con la ruta real a tu archivo .shp
shp_path = r'/ruta/completa/a/su/capa/barrios.shp'

# Creamos el objeto de la capa
# Parámetros: ruta, nombre de la capa en QGIS, proveedor de datos
layer = QgsVectorLayer(shp_path, "Barrios de Usaquén", "ogr")

# Verificamos si la capa es válida antes de añadirla
if not layer.isValid():
    print("¡Error! La capa no se pudo cargar correctamente.")
else:
    # Añadimos la capa al proyecto actual
    QgsProject.instance().addMapLayer(layer)
    print("Capa cargada exitosamente.")

2.2. Cargar Múltiples Capas Vectoriales

Para cargar varias capas a la vez, podemos crear una lista y recorrerla con un bucle for, lo cual es mucho más eficiente que hacerlo una por una.

from qgis.core import QgsVectorLayer, QgsProject

# Ruta a la carpeta que contiene todos los shapefiles
ruta_base = r'/ruta/completa/a/su/carpeta_de_shapes'

capas_a_cargar = [
    ("barrios.shp", "Barrios"),
    ("estacion_TM.shp", "Estaciones TM"),
    ("localidad.shp", "Localidad"),
    ("malla_vial.shp", "Malla Vial")
]

for archivo_shp, nombre_en_qgis in capas_a_cargar:
    ruta_completa = f"{ruta_base}/{archivo_shp}"
    layer = QgsVectorLayer(ruta_completa, nombre_en_qgis, "ogr")
    if layer.isValid():
        QgsProject.instance().addMapLayer(layer)
    else:
        print(f"No se pudo cargar la capa: {archivo_shp}")

2.3. Carga de Capas Ráster

Cargar capas ráster es muy similar, pero usamos la clase QgsRasterLayer y el proveedor "gdal".

from qgis.core import QgsRasterLayer, QgsProject

# Reemplaza con la ruta a tu archivo ráster (ej. .tif)
raster_path = r'/ruta/completa/a/su/raster.tif'

layer = QgsRasterLayer(raster_path, "Imagen Satelital Usaquén", "gdal")

if not layer.isValid():
    print("¡Error al cargar la capa ráster!")
else:
    QgsProject.instance().addMapLayer(layer)

3. Gestionando el Proyecto y las Capas

3.1. Guardar un Proyecto

Podemos guardar el estado actual de nuestro trabajo en un archivo de proyecto .qgs.

from qgis.core import QgsProject

# Ruta donde se guardará el proyecto
ruta_proyecto = r'C:\ruta\donde\guardar\mi_proyecto.qgs'

# Guardamos el proyecto
QgsProject.instance().write(ruta_proyecto)

3.2. Listar y Obtener Información de las Capas

Es muy útil poder listar las capas cargadas y acceder a sus propiedades.

from qgis.core import QgsProject

print("--- Capas actualmente en el proyecto ---")
for layer_id, layer in QgsProject.instance().mapLayers().items():
    tipo = "Vectorial" if layer.type() == QgsVectorLayer.VectorLayer else "Ráster"
    print(f"Nombre: {layer.name()}, ID: {layer_id}, Tipo: {tipo}")

Con el ID de una capa, podemos eliminarla del proyecto.

from qgis.core import QgsProject

# Reemplaza con el ID de la capa que quieres eliminar
id_capa_a_eliminar = "Barrios_Sacaidec_..." 

QgsProject.instance().removeMapLayer(id_capa_a_eliminar)

4. Carga de Capas desde PostGIS

Finalmente, podemos conectar QGIS a nuestra base de datos PostGIS y cargar capas directamente desde ella. Para esto, construimos una cadena de conexión (URI).

from qgis.core import QgsVectorLayer, QgsDataSourceUri, QgsProject

# Configuración de la conexión a la base de datos
uri = QgsDataSourceUri()
uri.setConnection("localhost", "5432", "nombre_db_bogota", "postgres", "su_contraseña")

# Especificar la tabla que queremos cargar
# Parámetros: esquema, tabla, columna de geometría, filtro (opcional), clave primaria
uri.setDataSource("public", "barrios", "geom", "", "id")

# Crear y añadir la capa
vlayer = QgsVectorLayer(uri.uri(False), "Barrios desde PostGIS", "postgres")
if vlayer.isValid():
    QgsProject.instance().addMapLayer(vlayer)
else:
    print("Error al conectar con la capa de PostGIS")

Dominar estas operaciones básicas en PyQGIS te permitirá automatizar tus flujos de trabajo, manejar grandes cantidades de datos con facilidad y extender las capacidades de QGIS para adaptarlas a tus necesidades específicas.
